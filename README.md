# Password Strength Analyzer

Cybersecurity project developed in Python.

## Features

- Password entropy calculation
- Crack-time estimation
- SHA-256 hashing demo
- Common password detection
- Leetspeak weakness detection
- Keyboard pattern detection
- Sequential character detection
- Corporate password policy checker
- Security recommendations
- Automatic report export

## Technologies

- Python
- Regex
- Cryptography Concepts

## Purpose

Built to strengthen understanding of authentication security concepts and password analysis techniques.
